# join-ui
### 灵集科技 UI补充组件库   
PC端  请使用ElementUI     
移动端 请使用 Vant 作为补充方案


## npm 安装
```
npm install join-ui
```

##  使用
```
// 按需引入
import {juiqrcode}  from 'join-ui'
Vue.use(juiqrcode)
// 完整引入
import jui from 'join-ui'
Vue.use(jui)
```
页面代码
```
<jui-qrcode text="http://www.nmgjoin.com"></jui-qrcode>
```
